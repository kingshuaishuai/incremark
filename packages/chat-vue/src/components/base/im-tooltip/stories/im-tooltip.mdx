# ImTooltip

提示组件，基于 `ImPopover` 实现，悬停触发，深色背景。

## 设计理念

`ImTooltip` 是对 `ImPopover` 的封装，专门用于显示简短的提示信息：
- 默认 hover 触发
- 默认显示箭头
- 深色背景样式
- 无内容时直接渲染子元素

## 特性

- 💡 简洁的 API
- 🎯 支持 12 种位置
- ➡️ 自带箭头
- 🌙 深色背景样式
- 🔄 无内容时降级为普通渲染


### CSS 类结构

- `.im-tooltip` - 提示内容容器

### 与 ImPopover 的区别

| 特性 | ImTooltip | ImPopover |
|------|-----------|-----------|
| 触发方式 | 仅 hover | hover/click/focus/manual |
| 箭头 | 默认显示 | 可选 |
| 样式 | 深色背景 | 无默认样式 |
| 使用场景 | 简短提示 | 复杂内容 |
