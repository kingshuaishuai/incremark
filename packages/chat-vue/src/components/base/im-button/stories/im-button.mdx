# ImButton

按钮组件，提供完整的设计系统支持。

## 设计系统

`ImButton` 支持：
- **6 种变体**: solid, outline, soft, subtle, ghost, link
- **5 种颜色**: primary, neutral, success, warning, error
- **5 种尺寸**: xs, sm, md, lg, xl

## 用法

```vue
<script setup>
import { ImButton } from '@incremark/chat-vue';
import { Icon } from '@iconify/vue';
import { SvgIcon } from '@incremark/chat-vue';
</script>

<template>
  <!-- 基础用法 -->
  <ImButton>Default</ImButton>

  <!-- 指定变体和颜色 -->
  <ImButton variant="outline" color="success">Success</ImButton>

  <!-- 带图标 -->
  <ImButton>
    <SvgIcon><Icon icon="ph:rocket" /></SvgIcon>
    Launch
  </ImButton>

  <!-- 仅图标按钮 -->
  <ImButton square>
    <SvgIcon><Icon icon="ph:plus" /></SvgIcon>
  </ImButton>
</template>
```
